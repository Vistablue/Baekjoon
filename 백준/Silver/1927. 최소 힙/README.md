### 문제 설명

<p>널리 잘 알려진 자료구조 중 최소 힙이 있다. 최소 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성하시오.</p>

<ol>
	<li>배열에 자연수 x를 넣는다.</li>
	<li>배열에서 가장 작은 값을 출력하고, 그 값을 배열에서 제거한다.</li>
</ol>

<p>프로그램은 처음에 비어있는 배열에서 시작하게 된다.</p>

### 입력 

 <p>첫째 줄에 연산의 개수 N(1 ≤ N ≤ 100,000)이 주어진다. 다음 N개의 줄에는 연산에 대한 정보를 나타내는 정수 x가 주어진다. 만약 x가 자연수라면 배열에 x라는 값을 넣는(추가하는) 연산이고, x가 0이라면 배열에서 가장 작은 값을 출력하고 그 값을 배열에서 제거하는 경우이다. x는 2<sup>31</sup>보다 작은 자연수 또는 0이고, 음의 정수는 입력으로 주어지지 않는다.</p>

### 출력 

 <p>입력에서 0이 주어진 횟수만큼 답을 출력한다. 만약 배열이 비어 있는 경우인데 가장 작은 값을 출력하라고 한 경우에는 0을 출력하면 된다.</p>

 ### 코드 
 ```python
import heapq
import sys

# 빠른 입출력을 위한 설정
input = sys.stdin.readline

n = int(input())
min_heap = [] # 일반 리스트를 생성하지만 heapq 함수로 제어하여 힙으로 운용

for _ in range(n):
    x = int(input())
    if x > 0:
        # 데이터 삽입 시, 최솟값이 맨 앞으로  (Heapify)
        heapq.heappush(min_heap, x)
    else:
        if not min_heap:
            print(0)
        else:
           # 가장 작은 값 추출 및 재정
            print(heapq.heappop(min_heap))
```
### 풀이
문제에서 첫째 줄에 연산의 개수 N(1 ≤ N ≤ 100,000)로 범위를 지정했기에 input()으로 쓰면 시간초과에러가 발생하므로 
sys.stdin.readline을 사용하였고 , 다른 언어와 달리 파이썬에서는 힙 라이브러리를 제공하기에 입력값이 양수일땐 heapq.heappush()를 호출하였고 이는 리스트 끝에 새원소를 삽입하고 부모노드와 비교하며 위로올리는 과정을거쳐 최솟값을 인덱스0 으로 올리는 작업을하였습니다. 입력값이 0인경우에는 힙이 비었으면 0을출력하고 , 아니라면 heapq.heappop()을 호출하여 인덱스 0번의 최솟값을 꺼내고 , 마지막 원소를 루트노드로 옮겨서 다시 자식노드들과 비교하며 내리는 과정을 거쳐 구조재정렬을 하도록하는 함수를 호출하였습니다.
### 배운점
파이썬의input() 함수를 사용했을 때 I/O 병목 현상으로 인한 시간 초과가있었는데 
sys.stdin.readline를쓰게되면 데이터를 버퍼에 한꺼번에 담아와서 처리하는 방식을 사용해 시스템 콜횟수를 줄여 데이터 로딩속도가 향상되 입출력 최적화가 이런 알고리즘 시간복잡도 문제에선 필수적임을 느꼈고,
자바론 힙을 구현하기 까다로웠었는데 파이썬에선 heapq를 쉽게 구현할수있단걸 이번문제에서 체감하였습니다.
